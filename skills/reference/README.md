# PPTX テンプレート置き場

## テンプレート（1ファイル）
- `d-template.pptx` … d Template 16:9。白背景・黒背景の両レイアウトを含む1ファイル。
  - ★配布元のサンプル/ガイドスライドは削除し、スライドマスターとレイアウトだけ残した版を置くこと。
  - 黒背景レイアウトは「投影専用」。配布・印刷では白背景を使う。

## アイコン
- `icons/` … アイコンライブラリ（D brand icon library）から、よく使うアイコンを
  PNG で書き出して置く。ファイル名はカテゴリ/用途が分かる名前にする（例：business-office.png）。
  - 枠を外しての使用は不可。元のアイコンをそのまま書き出すこと。

## メンテ
- レイアウトの index/名称が分かったら、../SKILL.md の「5. レイアウト対応表」を埋める。
- 免責・表紙の種別（白/黒・外部/内部・監査法人用）の対応も SKILL.md に追記する。
